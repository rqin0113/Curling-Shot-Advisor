# Curling Shot Advisor

An interactive curling strategy web app that analyzes the current stone positions on a rink and recommends the optimal next shot: **Draw**, **Guard**, or **Takeout**.

- **Backend:** Python + FastAPI  
- **Frontend:** JavaScript + HTML5 Canvas  
- **Features:**
  - Place blue and red stones interactively on a curling rink
  - Automatically calculates the best shot based on stone layout
  - Prevents overlapping stones and dynamically updates recommended green stone
  - Simulates a real curling rink for strategy practice

This project demonstrates full-stack development, algorithmic decision-making, and interactive data visualization.
